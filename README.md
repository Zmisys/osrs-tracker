# osrs-tracker

OSRS and OSRS Ironman specific progression tracking tool that provides recommendations, wiki quick access, and goals for new or returning players.

## 🌐 Live Demo

The application is hosted on GitHub Pages and can be accessed at:
`https://zmisys.github.io/osrs-tracker/`

## 🚀 Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. Any push to the `main` or `master` branch will trigger a new deployment.

### Manual Deployment

You can also trigger a manual deployment by:
1. Going to the "Actions" tab in the GitHub repository
2. Selecting the "Deploy to GitHub Pages" workflow
3. Clicking "Run workflow"

## 📁 Project Structure

- `index.html` - Main application page
- `.github/workflows/deploy.yml` - GitHub Actions workflow for automatic deployment

## 🛠️ Local Development

To run the application locally, simply open the `index.html` file in your web browser.

## 📝 Features

- 📊 Progress tracking for skills and achievements
- 🎯 Goal setting and tracking
- 📚 Quick access to OSRS Wiki
- 💡 Personalized recommendations
- ⚔️ Specialized support for Ironman accounts
- 🚀 Beginner-friendly interface
